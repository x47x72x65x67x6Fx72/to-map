# to-map (Toram Online - Interactive Map)

An interactive map for the MMORPG **Toram Online**. This project provides a static website that allows players to visually explore the game world's zones, their connections, and information on monsters and bosses.

## 🌟 Features

* **Interactive Visualization:** Visual representation of the different areas (Nodes) and their connections (Edges).
* **Detailed Information:** Display of zone details, available bosses, and regular monsters (mobs) for each area.
* **Lightweight & Fast:** As a static page, no server or database requests are necessary.

## 🛠️ Technologies

* HTML5 & CSS3
* Vanilla JavaScript
* [vis.js](https://visjs.org/) (for rendering the interactive network nodes)

## 🚀 Local Usage

Since the project is entirely static, running it is extremely straightforward:

1. Clone the repository to your machine:
   ```bash
   git clone https://github.com/your-username/to-map.git
   ```
2. Navigate into the downloaded project folder.
3. Open `index.html` (or the main HTML file) in any modern web browser.
